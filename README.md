# Phylex Platforms Dashboard

A modern dashboard built with Next.js, TypeScript, Tailwind CSS, and shadcn-ui.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Dark mode UI
- Responsive sidebar navigation
- Dashboard widgets (Account balance, Services, VAT, Business balance)
- Services table with filtering and pagination
- Built with shadcn-ui components

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- shadcn-ui
- Radix UI primitives
